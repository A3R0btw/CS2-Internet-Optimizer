# 🚀 CS2 Tweaks / Optimizer Suite

**Автор:** a3r0btw

---

## 📌 Описание

Этот набор скриптов предназначен для тонкой настройки сети и системы Windows с целью:

- уменьшения задержки (ping / input lag)
- оптимизации сетевого стека Windows
- настройки RSS (Receive Side Scaling)
- улучшения отклика в CS2

> ⚠️ Используется на свой страх и риск. Некоторые твики могут влиять на стабильность сети.

---

## ⚙️ Что входит в данный пак ди действия скриптов

### 🔧 FixRegistration.bat
- Отключает группировку пакетов (`InterruptModeration = 0`)
- Настраивает буферы приёма/передачи (1024)
- Отключает алгоритм Нейгла (`TcpNoDelay = 1`)
- Устанавливает `TcpAckFrequency = 1`, `TcpDelAckTicks = 0`
- Отключает энергосбережение сетевой карты
- Настраивает NetworkThrottlingIndex для снижения задержек
- Добавляет QoS политику для CS2 (DSCP 46)
- Настраивает Offload параметры

### 🧠 BindingRSSCores.bat
- Автоматически определяет количество ядер/потоков вашего процессора
- Привязывает обработку сетевых прерываний к **ПОСЛЕДНИМ ДВУМ** логическим ядрам
- Разгружает первые ядра для игры и снижает DPC latency

**Поддержка процессоров:**
- AMD Ryzen (8400F, 5600X, 7800X3D и любые другие)
- На Intel RSS применяется к последним энергоэффективным ядрам (в разработке)

### 🔄 ResetAllTweaks.bat
Сбрасывает все изменения:
- Параметры сетевых адаптеров (буферы, InterruptModeration)
- TCP настройки (TcpAckFrequency, TcpNoDelay, TcpDelAckTicks)
- NetworkThrottlingIndex и SystemResponsiveness
- QoS политику для CS2
- Привязку RSS (возвращает к стандартной)
- NetBT/NetBIOS на стандартные значения
- Winsock и TCP/IP стек

## 🖥️ Требования

- Windows 10 / 11
- Права администратора
- PowerShell (встроен)

---

## 🚀 Запуск

1. Для начала обновите драйвер интернета до **Последней версии**
2. Используйте **[DriverStoreExplorer](https://github.com/lostindark/DriverStoreExplorer/releases)** для удаления старых и конфликтующих драйверов (В паке он есть, версия )
3. ПКМ по `FixRegistration.bat` файлу, выбрать **Запуск от имени администратора** и дождаться завершение процесса 
4. ПКМ по `BindingRSSCoresX.bat` файлу, выбрать **Запуск от имени администратора** и дождаться завершение процесса 
      - В паке 2 файла `BindingRSSCores1.bat` и `BindingRSSCores2.bat`. Если вы стримите то рекомендую использовать `BindingRSSCores2.bat`, если только играете то используйте `BindingRSSCores1.bat`
5. Перезагрузить ПК **ОБЯЗАТЕЛЬНО**

---

## 🧠 RSS (важно)

Если RSS не применяется:

- проверь поддержку сетевого адаптера
- отключи Wi-Fi адаптеры
- убедись, что драйвер обновлён

---

## ⚠️ Возможные проблемы

- Скрипт может закрываться при ошибке PowerShell
- Некоторые NIC не поддерживают RSS
- Антивирус может блокировать изменения реестра

---

## 🛠️ Рекомендации

- Использовать Ethernet вместо Wi-Fi
- Обновить драйвер сетевой карты
- Перезагрузить ПК после применения
- Через программу **ProcessLasso** у cs2.exe снять галочки с 0 и **2 последних потоков ОБЯЗАТЕЛЬНО**
